# RayPI

RayPI是一个基于.NET Core 3.1的极简风Web开发框架。

目标：

* 极简，易读、易写、可扩展
* 适度封装，在实现功能的前提下，拒绝过度封装
* 领域驱动（DDD）：可以贫血也可以充血，视需求复杂度而定，当需求很简单时完全可以当成三层架构去写
* 支持读写分离（CQRS）
* 支持微服务架构

进度and计划：

* 搭出基础框架，：已完成
* 集成EFCore，简单实现CRUD：已完成
* 重构仓储模块和领域模块：进行中
* 重构AppService模块：进行中
* 重构配置模块，整合Options模式
* 重构日志模块
* 重构异常管理模块
* 待定...
